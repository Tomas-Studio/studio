## Getting Started

### Prerequisites

- Node.js 18+ or 20+
- pnpm 9.1.2+

### Installation

Install dependencies:

```bash
pnpm install
```

### Development

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

### Build

Build the application for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```

Generate static site:

```bash
pnpm generate
```
